# Tomás Verde - Reciclaje Orgánico en Pichilemu

Sitio web para Tomás Verde, un servicio de recolección y compostaje de residuos orgánicos en Pichilemu, Chile. El sitio está diseñado con un enfoque en la estética artesanal y ecológica, utilizando elementos orgánicos y naturales en su diseño.

## Características

- Diseño responsivo y moderno
- Animaciones orgánicas y naturales
- Paleta de colores inspirada en la naturaleza
- Tipografías artesanales y legibles
- Iconografía personalizada
- Optimizado para SEO

## Tecnologías utilizadas

- HTML5
- SCSS
- JavaScript
- GSAP (animaciones)
- Parcel (bundler)

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/tomasverde/website.git
cd website
```

2. Instala las dependencias:
```bash
npm install
```

3. Inicia el servidor de desarrollo:
```bash
npm start
```

4. Para compilar para producción:
```bash
npm run build
```
